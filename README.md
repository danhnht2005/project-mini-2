# 🚀 Project Mini 2 - Softy Pinko Landing Page Clone

Một dự án nhỏ (mini-project) tiếp theo nhằm mục đích rèn luyện và nâng cao kỹ năng xây dựng giao diện web (Frontend) thông qua việc "clone" (mô phỏng) lại trang chủ của mẫu giao diện **Softy Pinko** (từ TemplateMo).

## 🛠️ Công nghệ sử dụng
Dự án được xây dựng dựa trên các công nghệ web cơ bản kết hợp với CSS framework phổ biến:
- **HTML5:** Xây dựng cấu trúc ngữ nghĩa chuẩn SEO cho trang web.
- **CSS3:** Tuỳ chỉnh giao diện, màu sắc (gradient hồng/tím đặc trưng) và các hiệu ứng animation bắt mắt.
- **Bootstrap 4:** Ứng dụng hệ thống lưới (Grid system) linh hoạt và các component có sẵn để đẩy nhanh tốc độ dàn trang, đảm bảo trang web hiển thị responsive hoàn hảo trên mọi kích thước màn hình (Mobile, Tablet, Desktop).

---

## 📑 Các thành phần chính (Features/Sections)
Trang web là một cấu trúc Single Page (Landing Page) hoàn chỉnh, bao gồm các phần:
1. **Header & Hero Section:** Thanh điều hướng và phần giới thiệu chính với background gradient nổi bật.
2. **Features (Tính năng):** Các khối hiển thị tính năng của sản phẩm/dịch vụ theo dạng thẻ (cards) kết hợp icon.
3. **Work Process:** Khu vực trình bày quy trình làm việc từng bước trực quan.
4. **Testimonials:** Khối hiển thị đánh giá từ khách hàng/người dùng.
5. **Pricing Plans:** Bảng giá các gói dịch vụ (Basic, Premium, Advanced).
6. **Contact Us:** Biểu mẫu liên hệ (Contact form) và thông tin địa chỉ.
7. **Footer:** Chân trang với thông tin bản quyền và thiết kế đồ họa lượn sóng (wave) tạo điểm nhấn.
